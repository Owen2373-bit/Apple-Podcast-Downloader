This is an Apple Shortcut that allows you to download Apple Podcasts by using the share link. Just copy the share link and then run the Shortcut. Works with iPhone, Mac, iPad, etc.

**How to Download**

The easiest way to download this shortcut is by using the icloud link: https://www.icloud.com/shortcuts/1edf5180502e48b9b16949e41545bd0d 

You can also download it by clicking the file ending in .shortcut and then clicking "raw" or the download file botton depending on where you are viewing this github.

Ignore the other files in this github.

**How to Use It**

Simply run the shortcut on any Apple device and enter the name of the podcast then select where to download it. 

You can then access your downloaded file in Finder or Files.
